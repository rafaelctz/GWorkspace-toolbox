# Visão Geral dos Recursos

O GWorkspace Toolbox fornece três ferramentas poderosas para administradores do Google Workspace, cada uma projetada para resolver desafios administrativos específicos.

## Resumo de Recursos

| Recurso | Propósito | Casos de Uso |
|---------|-----------|--------------|
| [Extrator de Alias](/pt/features/alias-extractor) | Exportar todos os alias de usuário para CSV | Auditoria, migrações, relatórios |
| [Injetor de Atributos](/pt/features/attribute-injector) | Injetar atributos em lote para OUs | Campos personalizados, códigos de departamento, tags |
| [Sincronização de Grupos OU](/pt/features/ou-group-sync) | Sincronizar membros de OU para Grupos do Google | Gerenciamento automático de grupos, permissões |

## Extrator de Alias

Extrai todos os alias de e-mail do seu domínio Google Workspace e exporta-os para um arquivo CSV.

**Benefícios Principais:**
- Visibilidade completa de todos os alias de e-mail
- Exportação fácil para documentação ou migração
- Sem trabalho manual clicando em cada usuário
- Perfeito para auditorias de conformidade

[Saiba mais →](/pt/features/alias-extractor)

## Injetor de Atributos

Injeta atributos personalizados em lote para todos os usuários dentro de uma Unidade Organizacional.

**Benefícios Principais:**
- Atualiza centenas de usuários em segundos
- Aplica atributos consistentes em todas as equipes
- Sem necessidade de atualizações manuais usuário por usuário
- Suporta qualquer atributo de esquema personalizado

[Saiba mais →](/pt/features/attribute-injector)

## Sincronização de Grupos OU

Sincroniza automaticamente usuários de uma Unidade Organizacional para um Grupo do Google com capacidades de sincronização inteligente.

**Benefícios Principais:**
- Gerenciamento automático de associação de grupos
- A Sincronização Inteligente preserva membros adicionados manualmente
- Agende sincronização diária
- Reduza a manutenção manual de grupos

[Saiba mais →](/pt/features/ou-group-sync)

## Capacidades Comuns

Todos os recursos compartilham estas capacidades:

### Autenticação
- Autenticação OAuth 2.0 segura
- Sem armazenamento de senha
- Atualização automática de token
- Autenticação por sessão

### Suporte Multilíngue
- Interface completa em inglês, espanhol e português
- Troca de idioma sem interrupções
- Mensagens de erro e notificações localizadas

### Interface Moderna
- Design limpo e intuitivo
- Atualizações de status em tempo real
- Indicadores de progresso para operações longas
- Layout responsivo para dispositivos móveis

### Tratamento de Erros
- Mensagens de erro claras
- Retentar automático para falhas transitórias
- Registro detalhado para solução de problemas
- Degradação elegante
