# GWorkspace Toolbox

Bem-vindo ao GWorkspace Toolbox - um poderoso conjunto de ferramentas projetado para administradores do Google Workspace para otimizar operações diárias e tarefas de gerenciamento.

## Visão Geral

O GWorkspace Toolbox fornece uma interface moderna e fácil de usar para gerenciar operações do Google Workspace que de outra forma exigiriam scripts complexos ou processos manuais. Construído com eficiência e facilidade de uso em mente, ajuda os administradores a economizar tempo e reduzir erros.

## Recursos Principais

### Extrator de Alias
Exporta alias de usuário do seu domínio Google Workspace para formato CSV para auditoria, relatórios ou propósitos de migração.

### Injetor de Atributos
Injeta atributos personalizados em lote para Unidades Organizacionais (OUs), facilitando o gerenciamento de propriedades de usuário em todo o seu domínio.

### Sincronização de Grupos OU
Sincroniza automaticamente usuários de Unidades Organizacionais para Grupos do Google com capacidades de sincronização inteligente e opções de agendamento.

## Por que GWorkspace Toolbox?

- **Fácil de Usar**: Interface limpa e intuitiva não requerendo experiência técnica
- **Multilíngue**: Suporte completo para inglês, espanhol e português
- **Seguro**: Autenticação OAuth 2.0 com sua conta Google Workspace
- **Código Aberto**: Desenvolvimento gratuito e impulsionado pela comunidade
- **Auto-hospedado**: Execute em sua própria infraestrutura para controle completo de dados
- **Atualizações Automáticas**: Implantação Docker com Watchtower mantém você atualizado
